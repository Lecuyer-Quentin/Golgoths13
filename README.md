## Getting Started

```bash
# Clone the Repository:
git clone https://github.com/Lecuyer-Quentin/Golgoths13.git
cd Golgoths13

# Install Dependencies:
npm install

# Run the Development Server:
npm run dev
```

Open [http://localhost:3000] in your browser to view the application.
